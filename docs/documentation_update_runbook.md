# Runbook - Mise a jour de la documentation

Lorsqu une route, un script ou une convention change :

1. mettre a jour `README.md`
2. mettre a jour `docs/style-frontend-ui.md` si impact UI
3. mettre a jour `docs/metrics_catalog.md` si API/structure de donnees evolue
4. mettre a jour `docs/modifications.txt` pour tracer le changement
5. verifier que la documentation ne contient plus de reference au projet source
